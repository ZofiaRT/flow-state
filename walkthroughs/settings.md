# You Are in Control

Flow-State is packed with features to protect your mental energy, but **you don't have to use everything at once!** If you only want to track Code Complexity and ignore Reviewer Fatigue, you can easily toggle these features. We've included "Master Switches" for the Developer and Reviewer Cognitive Load Trackers, as well as granular controls for every single threshold (like the 120-second reading timer or the 400 LOC PR limit).

Tailor Flow-State exactly to your workflow and slowly enable new features as you get used to them.
