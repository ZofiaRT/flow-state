# Developer Cognitive Load Tracker

Burnout creeps up when you spend too much time struggling with complex code. Flow-State watches your back by analyzing:

- **Code Complexity:** Continuously evaluates the complexity of the code you are reading or writing to warn you if a file is getting too hard to understand.
- **Read/Write Ratios:** Warns you if you've been stuck reading without typing for too long.
- **Add/Delete Ratios:** Detects if you are heavily deleting code (a common sign of frustration).
- **Large Insertions:** Triggers a review load warning if you paste massive amounts of code at once.

All of these work together to keep your mind fresh.
